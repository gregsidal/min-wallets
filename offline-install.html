Minimal Ethereum Utilities
Download Offline Wallet Tools

The offline wallet tools can be used to create wallets and sign transactions on a device that is never connected to the internet. Keeping private keys quarantined on an offline device is a recommended way to protect them from online threats.

To start, download onto a thumb drive or SD card, move the file to an offline device, and unzip it there.

On the offline device, open offline/index.html and use the tools to manage wallets and sign transactions.

Signed transactions can be saved to a thumb drive or SD card, moved back to the online device, and safely broadcast to the Ethereum network.
